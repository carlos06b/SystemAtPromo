# SystemAtPromo

Sistema desenvolvido em Java para gerenciamento de promotores de vendas, com controle de dados e parte financeira.

## Sobre

Esse projeto foi criado com o objetivo de organizar o controle de promotores em uma empresa, permitindo cadastro, consulta e gerenciamento das informações, além de controle financeiro básico.

O sistema foi estruturado seguindo uma separação em camadas (Model, DAO, Controller e View), utilizando JDBC para conexão com o banco de dados MySQL.

## Funcionalidades

- Login de usuários
- Cadastro de promotores
- Consulta e busca por promotores
- Ativar e inativar promotores
- Controle financeiro básico

## Tecnologias utilizadas

- Java
- MySQL
- JDBC
- Swing
- Git/GitHub
